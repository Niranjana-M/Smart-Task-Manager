A modern and interactive **Task Management Web Application** built using Flask.
This app helps users manage tasks efficiently with a clean UI, dashboard analytics, and productivity features.

---

## 🚀 Features

* ➕ Add, delete, and complete tasks
* 📊 Dashboard with total, completed, and pending tasks
* 🔍 Search and filter tasks
* 📅 Due date tracking
* 📈 Progress bar for task completion
* 🎨 Modern UI with animations and dark theme
* ⚡ Smooth user experience

---

## 🛠 Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS, Bootstrap
* **Database:** SQLite
* **Version Control:** Git & GitHub

## 🚀 How to Run Locally

1. Clone the repository:

```
git clone https://github.com/YOUR-USERNAME/smart-task-manager.git
```

2. Navigate to project folder:

```
cd smart-task-manager
```

3. Install dependencies:

```
pip install flask
```

4. Run the app:

```
python app.py
```

5. Open in browser:

```
http://127.0.0.1:5000/
```
